    **Part A: Add manage_status parameter to run_single_iteration()**

    Per Pitfall 1 in RESEARCH.md: `run_single_iteration()` currently sets campaign status to "running" (line 124) and "completed" (line 232) unconditionally. For multi-iteration use, the loop wrapper must manage status instead.

    Add parameter `manage_status: bool = True` to `run_single_iteration()`.
    - When `manage_status=True` (default -- backward compatible): behave as now (set "running" on entry, "completed" on success, "failed" on error).
    - When `manage_status=False`: skip ALL status updates (running, completed, failed) -- the caller manages status.

    Specifically:
    - Line 124 `await self._store.update_campaign_status(campaign_id, "running")` -- wrap in `if manage_status:`
    - Line 232 `await self._store.update_campaign_status(campaign_id, "completed")` -- wrap in `if manage_status:`

    **Part B: Add run_campaign() method**

    Add `async def run_campaign(self, campaign_id: str, progress_callback: Callable[[dict], Awaitable[None]] | None = None) -> dict[str, Any]:` to CampaignRunner.

    Import at top of file: `from typing import Any, Callable, Awaitable` and `from orchestrator.engine.optimization_loop import check_thresholds, compute_improvement, is_converged, build_iteration_feedback, find_best_composite, TimeEstimator`.

    Implementation:
    1. Load campaign via `self._store.get_campaign(campaign_id)`.
    2. Set campaign status to "running" via `self._store.update_campaign_status(campaign_id, "running")`.
    3. Create `TimeEstimator` instance. Compute total_steps_per_iteration = 5 (generating, scoring, simulating, analyzing, checking).
    4. Initialize: `previous_results = None`, `previous_analysis = None`, `best_scores_history: list[dict[str, float | None]] = []`, `improvement_history: list[float] = []`, `observed_step_durations: list[float] = []`, `all_iteration_results: list[dict] = []`, `stop_reason = "max_iterations"`.
    5. Loop `for iteration in range(1, max_iterations + 1)`:
       a. Emit progress_callback `{"event": "iteration_start", "campaign_id": campaign_id, "iteration": iteration, "max_iterations": max_iterations, "eta_seconds": estimator.estimate_remaining(...) * 60}`.
       b. Call `result = await self.run_single_iteration(campaign_id=campaign_id, iteration_number=iteration, previous_iteration_results=previous_results, previous_analysis=previous_analysis, manage_status=False)`.
       c. Extract best composite: `best_composite = find_best_composite(result["composite_scores"])`.
       d. Append to `best_scores_history`.
       e. If `len(best_scores_history) >= 2`: compute improvement via `compute_improvement(best_scores_history[-1], best_scores_history[-2])`, append to `improvement_history`.
       f. Build feedback for next iteration: `previous_results = build_iteration_feedback(result, result["analysis"])`, `previous_analysis = result["analysis"]`.
       g. Append result to `all_iteration_results`.
       h. Emit progress_callback `{"event": "iteration_complete", "campaign_id": campaign_id, "iteration": iteration, "max_iterations": max_iterations, "best_scores": best_composite}`.
       i. Check thresholds if `campaign.thresholds`: `all_met, status = check_thresholds(best_composite, campaign.thresholds)`. Emit `{"event": "threshold_check", ...}`. If `all_met`: set `stop_reason = "thresholds_met"`, break.
       j. Check convergence if `len(improvement_history) >= 2`: `if is_converged(improvement_history)`: set `stop_reason = "converged"`, break.
    6. After loop: set campaign status to "completed" via `self._store.update_campaign_status(campaign_id, "completed")`.
    7. Emit progress_callback `{"event": "campaign_complete", "campaign_id": campaign_id, "stop_reason": stop_reason, "iterations_completed": len(all_iteration_results)}`.
    8. Return `{"campaign_id": campaign_id, "iterations": all_iteration_results, "stop_reason": stop_reason, "iterations_completed": len(all_iteration_results), "best_scores_history": best_scores_history, "improvement_history": improvement_history}`.
    9. Wrap the entire loop in try/except: on exception, set status to "failed", emit error event, re-raise.
